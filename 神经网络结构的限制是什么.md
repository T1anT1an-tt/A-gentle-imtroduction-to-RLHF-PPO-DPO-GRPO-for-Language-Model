# 神经网络结构的限制是什么

在 DPO 推导中，经常会看到这样一个前提：如果不考虑神经网络结构的限制，RLHF 的 KL 约束优化目标可以写出理论闭式解。这个前提的意思不是说真实模型能够直接达到该解，而是先在一个理想化的函数空间中分析最优策略应该长什么样。

现实中的策略模型并不是任意函数，而是由神经网络参数 $\theta$ 表示的 $\pi_\theta$。因此，理论最优策略 $\pi^*$ 和可训练模型 $\pi_\theta$ 之间天然存在差距。这个差距主要来自以下三类限制。

## 1. 表达能力限制：模型只能在假设空间内逼近最优解

理论推导中的 $\pi^*$ 可以被理解为一个不受参数规模限制的最优概率分布。只要奖励函数和参考策略给定，闭式解可以为每个候选回答分配理想概率。

但真实语言模型具有固定结构和有限参数量，例如 7B、13B 或 175B 参数。有限参数意味着模型只能表达某一类函数，这个可表达函数集合通常被称为假设空间（hypothesis space）。

因此，即使理论上的 $\pi^*$ 存在，它也不一定落在当前神经网络能够表示的假设空间中。训练的目标只能是找到一个参数配置 $\theta$，使 $\pi_\theta$ 尽可能接近 $\pi^*$，而不是保证完全等于 $\pi^*$。

简要地说：

```text
理论推导：在所有可能策略中寻找最优策略 pi*
实际训练：在当前神经网络能表示的策略集合中寻找较优的 pi_theta
```

这就是表达能力限制带来的第一层差距。

## 2. 自回归结构限制：语言模型必须按 token 分解生成

RLHF 和 DPO 的理论目标通常写成对完整回答 $y$ 的概率：

$$
\pi(y \mid x)
$$

这里的 $y$ 表示一整段回答。理论上可以把完整回答视为一个整体，并直接讨论它的概率。

但大语言模型实际采用自回归分解（autoregressive factorization）。也就是说，完整回答的概率会被拆成逐 token 的条件概率乘积：

$$
\pi_\theta(y \mid x)
= \prod_{t=1}^{T} \pi_\theta(y_t \mid x, y_{<t})
$$

这意味着模型不是一次性生成完整回答，而是在每个时间步根据提示词和已经生成的前缀预测下一个 token。

这种结构带来的限制是：理论最优策略可以从完整序列层面分配概率，而真实模型必须通过一系列局部条件概率来间接表达完整序列的概率。只要逐 token 分解的结构与理想完整序列分布之间存在不匹配，模型就很难完全实现理论闭式解。

## 3. 优化限制：闭式最优不等于训练过程能够到达

闭式解表示在理想条件下可以直接写出最优策略形式。但神经网络训练不是直接代入公式得到最优参数，而是通过梯度下降在高维非凸损失面上迭代更新参数。

实际训练会受到多种因素影响：

- 初始化参数不同，优化轨迹可能不同。
- 学习率、batch size、clip range 等超参数会影响收敛质量。
- 非凸优化可能出现局部最优、鞍点或训练震荡。
- 采样噪声和奖励模型误差会进一步放大优化不稳定性。

因此，即使 $\pi^*$ 在理论上给出了方向，训练过程也只能尝试把 $\pi_\theta$ 推向更优区域，并不能保证找到全局最优参数。

## 总结

“不考虑神经网络结构限制”是一种理论分析前提。它允许先在理想策略空间中推导最优形式，再讨论如何把该形式转化为可训练目标。

真实训练中，策略模型受到三类核心约束：

1. **表达能力限制**：$\pi_\theta$ 只能表示神经网络假设空间内的策略。
2. **自回归结构限制**：完整回答概率必须通过逐 token 条件概率间接表达。
3. **优化过程限制**：梯度下降不保证到达理论全局最优解。

这也是 DPO 推导中需要把理论最优策略 $\pi^*$ 重新参数化为真实可训练策略 $\pi_\theta$ 的原因。DPO 的关键价值不在于假设模型能够完美达到闭式解，而在于把偏好优化目标改写成可以直接训练语言模型的形式。
